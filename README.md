## Getting Started


### Prerequisites

Get API Keys

`.env` file:
```sh
ANTHROPIC_API_KEY=""
OPENAI_API_KEY=""
HF_ACCESS_TOKEN=""
COHERE_API_KEY=""
```

### Installation

```sh
cargo build
```

### API and Swagger Docs

1. Run the API

2. Visit http://localhost:8080/docs
